part_system_exists


Description

With this function you can check to see if the given particle system indexed exists in the game or not. Note that if the variable being checked is an uninitialised variable (that a particle systemwould otherwise have its index assigned to) this will throw an error.


Syntax:

part_system_exists(ind);

Argument Description
ind The index of the particle system to check for.


Returns:

Boolean


Example:

if !part_system_exists(global.Sname)
    {
    global.Sname = part_system_create();
    }

The above code checks to see if the particle system referenced in the global variable exists and if it does not it is created.
