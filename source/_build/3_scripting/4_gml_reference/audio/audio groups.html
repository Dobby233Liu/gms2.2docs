Audio Groups


Your audio files can be grouped together from the IDE - you can create audio groups from the Audio Groups window, which can be opened from the Tools menu at the top of the IDE, and assign your files to a specific group from the Sound Properties Window. These groups can then be loaded or unloaded from memory as well as have certain properties set for them using the following functions:

  1. audio_group_load
  2. audio_group_unload
  3. audio_group_is_loaded
  4. audio_group_load_progress
  5. audio_group_name
  6. audio_group_stop_all
  7. audio_group_set_gain
