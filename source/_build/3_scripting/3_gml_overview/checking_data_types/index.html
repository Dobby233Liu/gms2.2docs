Checking Data Types


The functions listed below can all be used for checking an input value to find out what kind of data type it is:


  • is_string
  • is_real
  • is_bool
  • is_array
  • is_ptr
  • is_int32
  • is_int64
  • is_vec3
  • is_vec4
  • is_matrix
  • is_undefined
  • is_nan
  • is_infinity
  • typeof
